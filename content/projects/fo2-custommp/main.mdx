Plugin to add a custom, more stable and reliable client-server multiplayer mode to FlatOut 2 with extra features such as 127 player lobbies and working AI

![preview 1](https://i.imgur.com/dan9Mms.png)

## Installation

- Make sure you have v1.2 of the game, as this is the only version this plugin is compatible with. (exe size of 2990080 bytes)
- Plop the files into your game folder, edit `FlatOut2CustomMP_gcp.toml` to change the options to your liking.
- Enjoy, nya~ :3

## Notes

- This plugin is currently only compatible with the vanilla game. Using it alongside Selo's Car Pack or similar mods will lead to crashes.
- You must have port 23756 forwarded on your router if you intend to host a game.
