# v1.26

- Added a warning to the start screen if required plugins weren't found
- Added a skin to the FO2 Splitter by teddyator 
- Added a new custom soundtrack option that tries to be a more modern version of the same style of music that the vanilla game has
- Moved the event tracks to the Single Race track type selection instead of its own separate menu
- Event tracks can now be played in Time Trial mode
- The soundtrack switcher is now fully customizable through `Config/Music.toml`, you can add or remove playlists as you wish
- The soundtrack switcher now allows you to change the playlist for menus, races, FO1 races and derby separately
