# v1.70

- Added a Best Placement display for each cup in Rally Mode, showing which cups you've passed
- Added a popup whenever you gain a life in Deathmatch Derby
- Added FlatOut 2 variants of all tracks with different lighting and prop placements, accessible via a new Change Track button in the track selection menu
- Added new cheats to unlock hidden content
- Bronze and Silver medals now count less towards your game completion stat than gold medals
- Carnage Mode event scores are now saved in a more unique way, reducing the chance of scores becoming mismatched or corrupted
- Changed some Carnage Mode target scores after player feedback
- Fixed derbies in multiplayer instantly ending if the timer is set to anything but 5 minutes
- Shuffled some of the Carnage Mode events around, making harder events show up later
