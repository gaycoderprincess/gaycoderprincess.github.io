Nearly pixel-perfect recreation of the NFS Most Wanted speedometer in Assetto Corsa

Recommended to be used with [my port of the game's handling](https://gaycoderprincess.github.io/project/assetto-mwphysics) for working speedbreaker, NOS and perfect starts

## Installation

- Make sure you have the latest Steam version of the game, as this is the only version this plugin is compatible with. (exe size of 22890776 bytes)
- Plop the files into your game folder.
- Enjoy, nya~ :3

## Features

- (With the Most Wanted physics mod) The NOS and speedbreaker indicators work as expected, and the needle also changes color in the perfect launch range
- Nearly pixel-perfect recreation of the HUD with all of its original textures intact, including the font and texture resolution
- The HUD can be recolored and re-skinned just like Most Wanted's gauge customizations using the included `.toml` file
- Zero other mods required, the mod runs without CSP or any HUD frameworks

## Known issues

- The HUD currently only displays correctly with a 16:9 aspect ratio
